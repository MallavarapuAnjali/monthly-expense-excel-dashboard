# monthly-expense-excel-dashboard
Excel dashboard for analysing and visualizing monthly expenses using pivot tables and charts.

## Project Overview
This project analyzes personal monthly expenses and visualizes spending patterns using an interactive Excel dashboard. The dashboard helps track spending habits, identify major expense categories, and monitor monthly financial trends.

The objective of this project is to demonstrate skills in **data analysis, data visualization, and dashboard design using Microsoft Excel**.
