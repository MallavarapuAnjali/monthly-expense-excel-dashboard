# Monthly Expense Dashboard - Project Description

## Objective

The objective of this project is to analyze and visualize personal monthly expenses using an interactive Excel dashboard. The project demonstrates how raw financial data can be transformed into meaningful insights using data analysis techniques.

---

## Dataset Description

The dataset contains records of personal expenses including:

- Date of transaction
- Expense category
- Amount spent
- Payment type
- Description of expense

This data is used to analyze spending patterns and create visual insights.

---

## Data Analysis Process

The project follows these steps:

### 1. Data Collection
Expense data is recorded and organized in a structured dataset.

### 2. Data Cleaning
The dataset is formatted to ensure consistency and accuracy.

### 3. Pivot Table Analysis
Pivot tables are used to summarize data and calculate:

- Total expenses
- Category-wise expenses
- Monthly spending trends

### 4. Dashboard Creation
Charts and visualizations are created to build an interactive dashboard that presents key financial insights.

---

## Dashboard Components

The dashboard includes the following visual elements:

- Expense distribution by category
- Monthly expense trends
- Summary of total expenses
- Visual charts for easy interpretation

---

## Key Insights

- Identify the categories where most money is spent
- Track spending trends over time
- Improve budgeting and financial planning
